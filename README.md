#### **软件介绍**

实现一个命令行文本计数统计程序。能正确统计导入的纯英文txt文本中的字符数，单词数，句子数。

#### 软件用法

命令模式： wc.exe [参数] [文件名]

wc.exe

输入1、2、3、4，以确定测试字符数、单词数、句子数或行数

输入文件名。

#### 文件列表

1.txt 为测试文本文档

1.jpg 为测试结果

0.1.cpp为空项目

0.2.cpp为基础功能源代码

0.3.cpp为扩展功能源代码
